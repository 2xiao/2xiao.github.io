# 目录大纲

数据结构与算法部分，将分为以下三个模块介绍：时间空间复杂度、算法、数据结构。

其中，算法分为：[动态规划](./algorithm/2_dynamic_programming.md)、贪心算法、二分查找、深度优先&广度优先、双指针、滑动窗口、位运算、递归、分治、剪枝、回溯、排序。

数据结构有：堆、栈、单调栈、链表、set & map、队列、数组、字符串、树、字典树、并查集、图。
