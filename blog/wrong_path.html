    # 那些年我走过的弯路

    总结了一下，女生最容易走的三个弯路：

    一个是圣母心，燃烧自己去让家人过得更好；

    一个是恋爱脑，为了男人改变自己的人生规划；

    一个是走捷径，想靠男人过上物质优渥的生活；

    不要再为父母活了，也不要再恋爱脑为男人牺牲了，更别想走捷径靠男人买房买车了。

    为自己活吧，去建设自己的人生金字塔，想要的生活还是得靠自己去创造！
